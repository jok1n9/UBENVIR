Existem dois logins e registos diferentes dependendo se são uma empresa ou um particular.
O registo particular é intuitivo e requer apenas que o número de telemóvel e o nif sejam numéricos.
Para login já existe uma conta criada na base de dados com os seguintes dados:
User: joking
Pass : joking123

Para registar uma empresa é preciso introduzir os dados sendo o número de telemóvel e o numero de conta bancaria numéricos. Após o registo é necessário aceder á página do admin para aceitar o registo.
Para dar login já existe a seguinte conta:
User:continente
Pass: continente123

Caso acedam ao login pela empresa, podem, ou entrar na página de administrador preenchendo o user com “admin@” e deixando a pass em branco.


Para adicionar produtos, basta preencher as informações dentro do questionário. E clicar em “add product”. Não existe reação há adicão de produtos, mas se tocar em back to homepage eles são adicionados.
